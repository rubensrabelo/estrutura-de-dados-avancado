# Set Project

## Descrição

Este projeto implementa uma estrutura de dados **Set (conjunto dinâmico de inteiros)** utilizando uma **Árvore Rubro-Negra (Red-Black Tree)**.


## Estrutura do Projeto

```bash
SetProject/
│
├── main.cpp
│
├── docs/
│   ├── COMPILATION.md
│   ├── MAIN_USAGE.md
│   └── TESTS.md
│
├── test/
│   └── tests.cpp
│
├── lib/
│   ├── include/
│   │   ├── Node.h
│   │   ├── RBTree.h
│   │   └── Set.h
│   │
│   └── src/
│       ├── Node.cpp
│       ├── RBTree.cpp
│       └── Set.cpp
│
├── README.md
└── Makefile
```
